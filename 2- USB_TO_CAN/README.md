# USB TO CAN

This firmware acts as a translator between the Raspberry Pi and the CAN bus.
* It receives USB messages from Raspberry Pi and convert them to CAN messages and send them on the bus
* It receives CAN messages from the bus and convert them to USB messages and send them to the Raspberry Pi
